Pricing Calculator Script README
==============================

Purpose
-------
Interactive calculator to compute product pricing based on costs, time, and desired margins. Calculates total cost, minimum price (15% margin), and recommended price (40% margin).

Usage
-----
Run the script directly:
python calculator.py

It prompts for inputs: material cost, time in minutes, hourly rate, packaging cost, extras.

Output
------
- Breakdown of costs.
- Suggested prices.
- Option to save to pricing_log.csv.

Files Used
----------
- pricing_log.csv: Appends quote details with timestamp.

Notes
-----
- Interactive; requires user input.
- Margins are fixed at 15% and 40%.
- For GUI use, may need modification to avoid inputs.
